# PedestrianBridge (Unofficial Compatibility Fork)

This repository is an unofficial compatibility fork of PedestrianBridge.

## Ownership And Attribution

- Original PedestrianBridge authors retain full credit for the original mod.
- This fork exists only to keep the mod working on current game versions.
- If the original maintainer resumes updates, this fork can be archived or redirected.

## Compatibility

- Target game version: Cities: Skylines v1.21.1-f4
- This fork includes compatibility updates in the KianCommons submodule.
